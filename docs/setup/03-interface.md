# Phase 3: The 3-Panel Law & Interface Design

## 3.1 The Layout Law
Every operational screen MUST adhere to the 3-panel horizontal structure:
1. **LEFT PANEL (Context):** Navigation, Filters, Steps.
2. **MAIN PANEL (Work):** Tables, Forms, Kanban, Canvas.
3. **RIGHT PANEL (Intel):** AI Reasoning, Thinking Logs, Approval Gates.

## 3.2 Main Dashboard Design
### KPI / Stats Row
- Global Operational Velocity %
- Pending Approvals count
- Current Burn Rate vs Budget

### Active Streams
- High-fidelity project cards.
- Visual "Health Rings" (Signal Orange for risk, Agency Emerald for health).
- One-click launch to execution plans.

### Next Best Actions (The Queue)
- Ranked list of "AI Proposals" requiring human review.
- Approve / Edit / Reject buttons.
- "All Clear" checkmark state when queue is empty.

## 3.3 Visual Aesthetics
- **Theme:** Digital Luxury / FashionOS.
- **Fonts:** Playfair Display (Headers), Inter (UI).
- **Radius:** Large 2.5rem - 3rem curves.
- **Depth:** Subtle glassmorphism and thin 1px borders.
